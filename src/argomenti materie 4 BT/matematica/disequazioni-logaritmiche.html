Le disequazioni logaritmiche

Le disequazioni logaritmiche sono disequazioni in cui uno (o entrambi) i lati coinvolgono un logaritmo.
Come le disequazioni esponenziali, sono utili nell'analisi di situazioni che implicano moltiplicazioni ripetute come nei casi di interesse (in economia, l'interesse è il prezzo del denaro insomma, cioè si tratta di un pagamento aggiuntivo pagato da un mutuatario a un investitore o prestatore oltre al rimborso dell'importo preso in prestito.) e nel dicadimento esponenziale (in fisica e matematica è la diminuizione di una quantità proporzionale al valore della stessa).

La chiave per lavorare con le disequazioni logaritmiche è il seguente fatto:

Se a > 1 e x > y, allora loga x > loga y.
Altrimenti, se 0 < a < 1, allora loga x < loga y.

Ovviamente, la base di un logaritmo non può essere 1 o negativa.
Ancora più importante, è vero anche il contrario:

Se a > 1 e loga x > loga y, allora x > y. Altrimenti, se 0 < a < 1, allora x < y

In altri termini più formali, la funzione logaritmica f(x) = loga x è monotonicamente crescente (aumentando x aumenta sempre la funzione f(x)) per a > 1, e monotonicamente decrescente (aumentando x diminuisce sempre la funzione f(x)) per 0 < a < 1.
