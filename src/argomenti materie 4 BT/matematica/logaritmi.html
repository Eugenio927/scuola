I logaritmi

Un logaritmo è l'inverso della funzione esponenziale.
Nello specifico, un logaritmo è la potenza alla quale un numero (la base) deve essere elevato per produrre un dato numero.

Per esempio, log2 64 = 6, perchè 26 = 64.
In generale, c'è la seguente definizione:

z è il logaritmo di y avente base x se e solo se xz = y.
logx y = z ⇔ x2 = y

In primo luogo, dobbiamo conoscere la struttura di base di un logaritmo ("logaritmo" viene abbreviato log per comodità/convenizone).
loga b = c può essere scritto come ac = b , dove a è la base, c l'esponente, e b l'argomento.
Anche log senza base non è altro che l'abbreviazione di log in base 10.
Sapendo questo possiamo finalmente studiare le proprietà dei logaritmi.
